# CHAINLIB
Chainlib is a digital library platform where writers publish their work and readers are able to access these work in a secure and transparent way. The library is built on a Blockchain technology which provides a secure and decentralized way to store and manage data.

The project file contains command (code) that gives instructions on how to set up the app and access the updated version. Chainlib has allowed a few contributors adding their quarter for the success of the project.

# Understanding The GitHub Actions Workflow File
   Github action workflow has the responsibility of automating the build process, ensuring the project builds correctly when changes are made.

# BUILD PROJECT
Build project specifies the workflow name. This project aims at building a dashboard wirh various features including analytics, earnings, feedback and notifications.

# HOW TO SET UP THE PROJECT 
1. PREREQUISITES: Make sure to you have npm ( node package manager ) version 10.9.2 or higher installed 
2. INSTALLATION 
• Clone the repository using git
• Install dependencies using npm
• Run the development server using npm
3. VIEW THE PROJECT 
Open a web browser and navigate to http://localhost:3000 to see the project in action
